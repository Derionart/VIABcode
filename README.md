# VIAB — Digital Agency Showcase

A high-performance, pixel-perfect digital agency landing page featuring an immersive, hardware-accelerated dark UI and an elegant serverless data pipeline.

🌐 **Live Demo:** [https://viabcode.com/](https://viabcode.com/)

---

## 🚀 Key Features

* **Advanced 3D Micro-interactions:** Fluid 3D smartphone element tilt animation powered by the browser's `DeviceOrientation API` (accelerometer) and synchronized smooth rendering via `requestAnimationFrame`.
* **Serverless Backend Infrastructure:** Dynamic client contact form processing handled completely through asynchronously bound *Google Apps Script Web Apps* (eliminating server overhead and keeping infrastructure free & secure).
* **Environment Configuration Isolation:** Decoupled `config.js` design patterned to mirror production/development configuration setups (hidden securely from version control).
* **Production-Ready SEO:** Semantic layout tags, custom SVG assets, and automated XML mapping optimized for instant search engine indexing.

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3 (Custom Variables, CSS Gradients, 3D Transforms), Vanilla JavaScript (ES6+).
* **Backend Pipeline:** Google Apps Script / Google Sheets API.
* **Hosting & Automation:** Production-optimized routing, custom domain binding, and SSL architecture.
